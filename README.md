

### PROCESO DE INSTALACION
Instala los archivos necesarios
```bash
- pkg upgrade && update
- pkg install nodejs -y
- pkg install git -y
- pkg install bash -y
```

Clona este repositorio
 ```bash
> git clone https://github.com/NegK01/Beta23
```

Inicia la instalacion
```bash
- cd Beta23
- bash install.sh
```
Nota: Si la instalacion sera en Pc, utiliza
```bash
- bash installpc.sh
```

Para Iniciar el Bot
 ```bash
- npm start
```
